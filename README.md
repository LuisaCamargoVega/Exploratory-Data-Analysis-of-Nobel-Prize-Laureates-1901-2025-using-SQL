# Exploratory-Data-Analysis-of-Nobel-Prize-Laureates-1901-2025-using-SQL
This project presents an Exploratory Data Analysis (EDA) of Nobel Prize laureates from 1901 to 2025 using SQL.  The analysis explores historical trends, category distributions, gender representation, collaboration patterns, and the institutions and countries most associated with Nobel Prize winners.

Objectives
*Explore the structure of the dataset
*Analyze Nobel Prize distribution by category
*Examine gender representation among laureates
*Identify historical trends over time
*Apply SQL queries for filtering, grouping, and aggregation
*Extract meaningful insights from the data

Tools Used
*SQL
*RunSQL
*Kaggle Dataset

Key Insights
*Scientific categories dominate Nobel Prize awards
*Collaborative research has increased over time
*Nobel recognition is concentrated in a small group of countries and institutions
*Female representation has improved over time, although differences across fields still exist

Dataset

*Source: Kaggle — Nobel Prize Laureates (1901–2025)
